To make XMPP work on this server, you need to set up SRV records for {{ host }} domain. It will allow XMPP clients to correctly find the location of your server using Domain Name System (DNS). You need to add several types of DNS records to make things work:

  • A record for subdomain xabber.{{ host }} to make server management panel available from the Internet on this address
  • SRV records to help XMPP clients and servers find your server
